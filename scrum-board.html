⛔ Invariant 1 - never violate

The SEO / privacy island shell is sacred

Marketing, how-it-works, FAQ, and JSON-LD render at build time as static HTML with zero JS shipped. No file bytes ever leave the device - no fetch/XHR of PDF contents, no server, CSP connect-src 'self'. Off the table: SSR, moving marketing into JS, any external connect-src.

⛔ Invariant 2 - never violate

The gesture hot path stays as it is

During drag/resize, mutate the DOM directly for 60fps feedback; commit React state exactly once on pointerup. Never route continuous pointer events through reactive state or a store - that is the reconciliation thrash. Applies equally to drag and resize.
